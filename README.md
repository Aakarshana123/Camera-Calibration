# Camera-Calibration
Camera calibration is an essential task in computer vision that determines the relationship between 3D world coordinates and 2D image coordinates. This project focuses on estimating intrinsic and extrinsic parameters of a camera using calibration techniques. Standard methods such as Zhang’s calibration method are studied and implemented. The project also explores the effect of different camera poses and noise on calibration accuracy. Initial results show that multiple viewpoints improve accuracy, and further optimization is planned to enhance the performance.
