This repository contains the implementation of a camera calibration project titled “Best Poses for Camera Calibration.” The project focuses on estimating intrinsic parameters (focal length, principal point, and distortion coefficients) and extrinsic parameters (rotation and translation) using multiple checkerboard images captured from different viewpoints.

The implementation is carried out in MATLAB using standard calibration techniques, including Zhang’s method and built-in calibration functions. The project also analyzes the impact of camera pose on calibration accuracy by evaluating reprojection error for each image.

Key features of the repository include checkerboard corner detection, parameter estimation, distortion analysis, reprojection error visualization, and pose quality evaluation. The results demonstrate that images captured from diverse angles and orientations significantly improve calibration accuracy.
